# Scripts for running CycleViz and interpreting the results.

## Contents
`[Sample_ID]/run-cv.sh` Run CycleViz on AmpliconSuite result files for a given sample.  
`cycle_lengths.ipynb` Calculate base pair lengths of amplicons.  

## Requirements
environment: [py3](env/py3)  
[CycleViz](https://github.com/AmpliconSuite/CycleViz) v0.2.1  
[oscutils](https://github.com/auberginekenobi/oscutils) @adc5b13 or later  
AmpliconSuite results available on [AmpliconRepository](https://ampliconrepository.org/project/69c59d0cd69472646656266f)
